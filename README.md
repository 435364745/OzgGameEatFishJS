OzgGame—大鱼吃小鱼

================

需要将cocos2d-js的frameworks目录复制到OzgGameEatFishJS下，即OzgGameEatFishJS/frameworks，可用cocos new -l js WebGame命令得到该目录。

运行命令：cocos run -p web

目前遇到的问题：

游戏界面右上角的关卡和分数会自动改变位置，原因不明。
